## Wallpaper

Install the softwrare feh to set the wallpaper

```bash
sudo pacman -S feh
```

Set the wallpaper

```bash
feh --bg-scale /path/to/image.jpg
```

Configure your **.xprofile** file to set your background when you boot your PC
